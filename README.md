# Lazy A

A cinematic portfolio built as a physical place inside a browser.

## Vision
The goal is not to build a website.
The goal is to build a room.

Visitors borrow a point of view inside a believable creative space.

## Principles
- The room is real.
- The camera is a person.
- The impossible is earned.
- Reward attention, not clicks.
- Build one interaction at a time.
